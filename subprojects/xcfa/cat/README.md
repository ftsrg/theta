## Further reading

* [Handling axiomatic memory models in abstraction-based model checking of concurrent and distributed systems](https://theta.mit.bme.hu/publications/bajczilMsc2022.pdf) (MSc, 2022) — memory modelling for weakly ordered memory models, extended to message-based communication protocols.
* [Reasoning with Happens-Before Relations About Concurrent Programs in the THETA Framework](https://doi.org/10.1109/DSN-W65791.2025.00037) (VERDI 2025) — bounded model checking approaches reasoning with happens-before relations (implemented in `common/analysis`'s `algorithm/oc`).
* [On Stability in a Happens-Before Propagator for Concurrent Programs (Reproducibility Study)](https://doi.org/10.1007/978-3-031-90643-5_1) (TACAS 2025) — a correctness issue in the published happens-before propagator algorithm, and a revised version.

The full publication list is at [theta.mit.bme.hu/publications](https://theta.mit.bme.hu/publications/).
